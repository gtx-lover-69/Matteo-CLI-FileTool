# Matteo-CLI-FileTool
A CLI file opener, reader, and editor written in Python

Created without the help of AI, one of the projects I'm more proud of.

All you have to do is extract the zip file, and run the .exe inside.
When prompted, type in a number to make your selection. 

The .py file is source code and requires the libraries "colorama" and "pygments" to run.
